# BatchRename

## About
Think about if you download your favourite comics (or some art assets for your game dev) and get some images in one folder, but find them in bad naming rule, then you will be eager to rename them all in your way, therefore this tool aims at renaming enumerous files in a unified format you defined

## Usage
Just run this script in command line where Python interpreter environment has been set up, and follow the instructions to start your renaming

```bash
python Main.py
```
